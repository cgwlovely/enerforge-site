Project note 04 Project finance Working notebook

The problem is not technology. The problem is bankability.

In many early industrial energy projects, people start with the technology: solar size, battery duration, inverter selection, container layout, grid-forming capability.

Those are important. But they are not enough.

The harder question is usually not “can this system be built?” The harder question is:

Can this project be paid for, contracted and financed?

A battery can be technically useful and still fail commercially. A mine can save diesel cost and still not provide enough stable revenue to the project company. A financial model can show upside, but a lender may still ignore that upside if it is not contracted.

That is why I think early-stage BESS feasibility should connect five things:

  • Technical fit
  • Customer avoided cost
  • Contract revenue
  • Carbon value
  • Bankability

If these are assessed separately, the project can look better than it really is.

If they are assessed together, the next decision becomes clearer:

Proceed, optimise, redesign or stop.
